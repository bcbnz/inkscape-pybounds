BoundingBox¶

class bounds.BoundingBox(x0, x1, y0, y1)¶

A class which represents a bounding box. It has four attributes (left, right, bottom and top) which define the edges of the box, and a number of functions to work with the box.

Parameters:
  • x1 – The x-value representing one vertical edge of the box.
  • x2 – The x-value representing the other vertical edge.
  • y1 – The y-value representing one horizontal edge of the box.
  • y2 – The y-value representing the other horizontal edge.

When creating the box, the constructor will automatically choose the lower of the two x-values for the left edge and the higher for the right. Similarly, the lower y-value is used for the bottom of the box and the higher for the top of the box.

Attributes

left¶

The left-hand edge of the bounding box.

right¶

The right-hand edge of the bounding box.

bottom¶

The bottom edge of the bounding box.

top¶

The top edge of the bounding box.

Methods

combine(box)¶

Combines this box with another bounding box. This extends the edges of this box as necessary to encompass the contents of the other box.

Parameters:
  • box – The other box.
contains(point)¶

Check if the given point is contained with this box.

Parameters:
  • point – The point to check specified as a pair of numbers (x,y).
Returns:

True or False.
contains_x(x)¶

Check if the given x value is within the range of x values encompassed by the box.

Parameters:
  • x – The x value to check.
Returns:

True or False
contains_y(y)¶

Check if the given y value is within the range of y values encompassed by the box.

Parameters:
  • y – The y value to check.
Returns:

True or False
extend(point)¶

Extend the box as necessary to encompass the given point.

Parameters:
  • point – The point specified as a pair of numbers (x,y).
extend_x(x)¶

Extend the width of the box as necessary to include the given x point.

Parameters:
  • x – The x point to include.
extend_y(y)¶

Extend the height of the box as necessary to include the given y point.

Parameters:
  • y – The y point to include.
