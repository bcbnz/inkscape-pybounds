Helper functions¶

draw_bounding_box¶

bounds.draw_bounding_box(obj, style=None, replace=False)¶

Draws the bounding box of the given object.

Parameters:
  • obj – The XML node representing the object.
  • style – The SVG style to draw the bounding box with.
  • replace – Whether to replace the object with its bounding box.

This function draws the bounding box around the given object. If no style is specified, the style of the object is used to draw the bounding box. If replace is True, the object is removed from the image and is replaced by its bounding box. If it is False, the bounding box is drawn on top of the object.
